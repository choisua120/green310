제목태그

제목 태그는 h1~h6까지 숫자가 작아질 수록 글자가 커지며 특성도 커진다

제목태그-h1

제목태그-h2

제목태그-h3

제목태그-h4

제목태그-h5
제목태그-h6

제목태그의특징
- 글씨크기가 다르다(h3의 크기가 1입니다.)
- 볼드효과를 가진다
- 자동줄바꿈기능이 있음

주의사항
- h1은 주로 로고를 쓸떄 사용하므로 문서 내부에서는 1번만 사용하기를 권한다
- 자동줄바꿈의 기능을 블럭이라고 함
- 블럭내부의 블럭을 담지 못함

단락태그

문장과 같은 텍스트 요소들을 가두는 역할

p태그 특성
- 자동 줄바꿈 기능
- 위아래 여백을 가지고 있음

주의사항
- 블럭 내부에 블럭을 담지 못함

줄바꿈태그

br태그
- 문맥상 강제적으로 줄바꿈늘 하고 싶을 때 사용, 닫힘 태그 없음\

구분석 태그

hr태그
- 구분선을 그리고 싶을 때 사용, 화면의 가로시작부터 끝까지 가득 채움, 닫힘 태그 없음

링크태그

a태그
- 지정된 주소로 보내는 태그로 태그 내부에 사용하며 지정주소속성능을 사용한다.

href 속성 = "지정주소설정" 예) href=http: taget속성 = "_balnk" 새창 열림
속성은 여러개를 쓸 수 있으며 띄어쓰기로 다음 속성을 알린다

a태그 특성
- 밑줄 생성
- 색깔이 파란색으로 나타남
- 모든 태그 내부에 작성하며 특별한 줄바꿈이 일어나지않음/
단독적으로 사용가능하나 블럭태그를 담을 수 없음(인라인 태그 속성)

글자속성태그

텍스트관련요소로 강조,최소,첨자 등을 표현
모든 태그 내부에 첨부가 가능한 일렬 방식 (inline방식)

b태그는 단순 보여짐 효과, strong은 중요하다라는 의미 부여
